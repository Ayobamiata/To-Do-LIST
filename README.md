A simple backend-only RESTful To-Do API built using Flask.
This project focuses on core backend concepts such as routing, REST principles, JSON handling, and basic error management.

Features
Create, read, update, and delete tasks (CRUD)
JSON-based API responses
Basic error handling (404 for non-existent tasks)

Tested using Postman

Tech Stack
Python
Flask
Postman (for API testing)
